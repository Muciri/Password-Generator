## Como rodar

1. Clone o repositório em sua máquina:
   ```bash
   git clone https://github.com/seu-usuario/seu-repo.git

2. Acesse a pasta do projeto
    ```bash
    cd nome-do-projeto

3. Instale as dependências
    ```bash
    npm install

4. Inicie o aplicativo
    ```bash
    npm run dev
